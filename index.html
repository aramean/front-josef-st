Hi! I'm Josef Gabrielsson. I have a strong passion for frontend, DevOps, and backend development. I create dynamic and responsive user interfaces using HTML, CSS, and JavaScript, and I work with modern frontend frameworks such as React, Svelte, Angular - as well as my own low-code framework, Front, which enables rapid prototyping and scalable development.

On the backend, I'm proficient in C#, Python, PHP, Golang, Node.js, databases, server management, and RESTful API integration. I also have experience with Java and Android development.

I'm known for quickly learning new technologies, adapting to change, and maintaining a strong work ethic. I stay up to date with the latest trends across the entire development stack and continually develop my skills.

This CLI-style website is built using Front, a lightweight low-code framework I created, together with FTML, a custom markup language for interactive UI logic. It demonstrates how Front can build dynamic web interfaces using simple declarative syntax while keeping full control in the DOM.
